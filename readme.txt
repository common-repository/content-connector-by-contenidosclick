Conecta la plataforma de Contenidos Click con el blog del cliente, facilitando la publicación o puesta en borrador de los contenidos realizados.
Nota: Solo funciona siendo cliente activo de Contenidos Click.

== Description ==
Plugin para clientes de Contenidos Click que sólo funciona contratando previamente el servicio de redacción de contenidos.
El plugin conecta la plataforma de gestión y redacción de contenidos con el blog del cliente y de esta forma los contenidos quedan publicados automáticamente o en borrador según la opción deseada.

== Installation ==
Descargar el plugin e introducir los datos de conexión facilitados por Contenidos Click.

== Frequently Asked Questions ==
= ¿Es posible utilizar el plugin sin ser cliente de Contenidos Click? =

No, pero puedes contactar a cc@contenidosclick.es para más información.

= ¿Cómo lo hago para conectarme? =

Solicitando los datos de conexión a Contenidos Click

= ¿Puedo tener más de una web o blog con el plugin instalado? =

Si pero debes de insertar los códigos específicos para cada conexión.
